## 三、手把手教你启用 Passkeys

下面直接上实操，我以 **Google 账户**为例，其他平台大同小异。

### 在手机上设置

**iPhone 用户：**

1. 打开「设置」→ 你的名字 →「登录与安全」→「通行密钥」

2. 点击「添加通行密钥」

3. 系统会调用 Face ID 或指纹验证

4. 完成！以后登录 Google 就可以直接扫脸了

**Android 用户：**

1. 打开「设置」→「Google」→「管理您的 Google 账户」

2. 点击「安全性」→「通行密钥」

3. 点击「创建通行密钥」

4. 使用指纹或屏幕锁验证

5. 搞定！

### 在电脑上设置

**Windows 用户：**

1. 用 Chrome 或 Edge 打开 https://myaccount.google.com/security

2. 找到「您可以用其他方式登录 Google 账户」部分

3. 点击「通行密钥」→「创建通行密钥」

4. 使用 Windows Hello（指纹或面部识别）验证

5. 完成！

⚠️ **注意**：Windows 需要支持 Windows Hello 的设备（带指纹识别器或红外摄像头的笔记本/台式机）

## 五、常见问题解答

**Q1：手机丢了怎么办？**

Passkeys 会同步到你的云端（iCloud 或 Google 账号），换新手机直接恢复就行。不过建议**多设备备份**，别只留一个。



**Q2：不支持 Passkeys 的网站怎么办？**

目前还在过渡期，很多网站还不支持。这种时候还是得用传统密码，或者用密码管理器（如 1Password、Bitwarden）。



**Q3：安全吗？**

**超安全！**Passkeys 使用公钥加密，私钥永远不离开你的设备。就算网站数据库被黑，黑客也拿不到你的凭证。



**Q4：2026年密码会完全消失吗？**

不会马上消失，但趋势很明显。越来越多网站会默认支持 Passkeys，密码慢慢变成备选方案。
