## 模式说明

- **Planning（规划）**：

Agent 在执行任务之前会先进行规划，适用于深度研究、复杂任务或协作工作；



- **Fast（快速）**：

Agent 将直接执行任务，适用于简单任务，可以更快速地完成任务。



## 设置语言中文

快捷键 `Ctrl+Shift+P(Windows)` 输入

```
Configure Display Language
```

## 更换自己喜欢的主题

快捷键 `Ctrl+Shift+P(Windows)` 输入

```
Themes: Color Theme
```

点击“首选项:颜色主题”



[导入 Cursor自带主题](https://m.okjike.com/originalPosts/691e6c5100c0686ab5271356)
