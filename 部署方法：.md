## 部署方法：

1. 克隆项目

```
git clone https://github.com/afoim/fuwari
```

2. 安装依赖

```
cd fuwari
pnpm install
```

3. 修改配置

```
编辑 src/config.ts 和 astro.config.mjs 中的 site 地址
```

4. 本地构建测试

```
pnpm build
pnpm preview
```

5. 部署到 Cloudflare Pages

```
方式A: 通过 Cloudflare dashboard 连接 GitHub 仓库自动部署
方式B: 使用 wrangler CLI
npx wrangler pages deploy dist
```

```
 如果直接上传 dist 目录到Cloudflare  后续更新文章应该怎么做呢
```
